[![Dependency Status](https://deps.rs/repo/github/JohnScience/wasm_mouse/status.svg)][`wasm_mouse`/dep_status]

Making mouse events management in [`WASM`] with [`web-sys`] easier.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
wasm_mouse = "0.1"
```
